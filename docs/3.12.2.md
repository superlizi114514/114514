### 1. D1 SQLite 兼容性修复（已完成）
修复了 `db-vercel.ts` 中所有 `RETURNING *` 语法问题，因为 SQLite 不支持此语法。

- `createUser` - 使用 `lastInsertRowId` 获取新 ID 后查询
- `createProfile` - 同上
- `updateProfile` - 改为先更新再查询
- `createProfileReview` - 使用 `lastInsertRowId` 获取新 ID 后查询
- `createMerchant` - 使用 `lastInsertRowId` 获取新 ID 后查询
- `updateMerchant` - 改为先更新再查询
- `createMerchantReview` - 使用 `lastInsertRowId` 获取新 ID 后查询
- `createPartTimeJob` - 使用 `lastInsertRowId` 获取新 ID 后查询
- `updatePartTimeJob` - 改为先更新再查询
