# 📧 山信黑红榜 — 邮箱登录 + 图文验证方案

---

## 一、方案概述

**登录流程：**
```
1. 用户输入邮箱
2. 显示图形验证码（防机器人）
3. 输入验证码 + 图形验证码
4. 发送邮箱验证码（6 位数字）
5. 输入邮箱验证码登录
6. 自动注册（首次登录）
```

**成本：** ¥0/月（使用免费邮件服务）

**防刷机制：**
- 图形验证码防机器注册
- 邮箱验证防虚假账号
- 设备指纹 + IP 限流
- 同一邮箱每日最多 3 次验证

## 四、API 设计

### 4.1 获取图形验证码

```typescript
// GET /api/captcha

import { Canvas, Image } from '@napi-rs/canvas'

async function handleGetCaptcha(request: Request, env: Env) {
  const sessionId = crypto.randomUUID()
  
  // 生成 4 位随机码
  const code = Math.random().toString().slice(-4).toUpperCase()
  
  // 存入数据库（5 分钟过期）
  const expireAt = new Date(Date.now() + 5 * 60 * 1000)
  await env.DB.prepare(`
    INSERT INTO captcha_codes (session_id, code, expire_at)
    VALUES (?, ?, ?)
  `).bind(sessionId, code, expireAt.toISOString()).run()
  
  // 生成验证码图片
  const canvas = new Canvas(120, 40)
  const ctx = canvas.getContext('2d')
  
  // 背景
  ctx.fillStyle = '#f0f0f0'
  ctx.fillRect(0, 0, 120, 40)
  
  // 干扰线
  for (let i = 0; i < 3; i++) {
    ctx.strokeStyle = `rgb(${Math.random()*255},${Math.random()*255},${Math.random()*255})`
    ctx.beginPath()
    ctx.moveTo(Math.random() * 120, Math.random() * 40)
    ctx.lineTo(Math.random() * 120, Math.random() * 40)
    ctx.stroke()
  }
  
  // 文字
  ctx.font = 'bold 24px Arial'
  ctx.fillStyle = '#333'
  ctx.textAlign = 'center'
  ctx.textBaseline = 'middle'
  
  // 每个字符稍微偏移
  const chars = code.split('')
  chars.forEach((char, i) => {
    const x = 20 + i * 25
    const y = 20 + (Math.random() - 0.5) * 10
    ctx.save()
    ctx.translate(x, y)
    ctx.rotate((Math.random() - 0.5) * 0.3)
    ctx.fillText(char, 0, 0)
    ctx.restore()
  })
  
  // 干扰点
  for (let i = 0; i < 50; i++) {
    ctx.fillStyle = `rgb(${Math.random()*255},${Math.random()*255},${Math.random()*255})`
    ctx.fillRect(Math.random() * 120, Math.random() * 40, 2, 2)
  }
  
  // 返回图片
  const buffer = canvas.toBuffer('image/png')
  const base64 = buffer.toString('base64')
  
  return Response.json({
    success: true,
    sessionId,
    image: `data:image/png;base64,${base64}`
  })
}
```

### 4.2 发送邮箱验证码

```typescript
// POST /api/send-code

async function handleSendEmailCode(request: Request, env: Env) {
  try {
    const { email, captchaCode, captchaSession } = await request.json()
    
    // 验证邮箱格式
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/
    if (!emailRegex.test(email)) {
      return Response.json({ success: false, message: '邮箱格式错误' })
    }
    
    // 验证图形验证码
    const captcha = await env.DB.prepare(`
      SELECT * FROM captcha_codes 
      WHERE session_id = ? AND code = ? AND is_used = 0 AND expire_at > ?
    `).bind(captchaSession, captchaCode.toUpperCase(), new Date().toISOString()).first()
    
    if (!captcha) {
      return Response.json({ success: false, message: '图形验证码错误' })
    }
    
    // 标记图形验证码已使用
    await env.DB.prepare(`
      UPDATE captcha_codes SET is_used = 1 WHERE id = ?
    `).bind(captcha.id).run()
    
    // 检查发送频率（同一邮箱每日最多 3 次）
    const today = new Date().toISOString().slice(0, 10)
    const todayCount = await env.DB.prepare(`
      SELECT COUNT(*) as count FROM email_codes 
      WHERE email = ? AND DATE(created_at) = ?
    `).bind(email, today).first()
    
    if (todayCount.count >= 3) {
      return Response.json({ 
        success: false, 
        message: '今日发送次数已达上限，请明日再试' 
      })
    }
    
    // 生成 6 位邮箱验证码
    const code = Math.random().toString().slice(-6)
    const expireAt = new Date(Date.now() + 10 * 60 * 1000)
    
    // 存入数据库
    await env.DB.prepare(`
      INSERT INTO email_codes (email, code, expire_at)
      VALUES (?, ?, ?)
    `).bind(email, code, expireAt.toISOString()).run()
    
    // 发送邮件
    await sendEmail(email, code, env)
    
    return Response.json({ 
      success: true, 
      message: '验证码已发送，请检查邮箱' 
    })
  } catch (e) {
    console.error(e)
    return Response.json({ success: false, message: '发送失败' })
  }
}
```

### 4.3 邮箱验证码登录

```typescript
// POST /api/login

async function handleLogin(request: Request, env: Env) {
  try {
    const { email, code } = await request.json()
    
    // 验证邮箱验证码
    const result = await env.DB.prepare(`
      SELECT * FROM email_codes 
      WHERE email = ? AND code = ? AND is_used = 0 AND expire_at > ?
    `).bind(email, code, new Date().toISOString()).first()
    
    if (!result) {
      return Response.json({ 
        success: false, 
        message: '验证码错误或已过期' 
      })
    }
    
    // 标记验证码已使用
    await env.DB.prepare(`
      UPDATE email_codes SET is_used = 1 WHERE id = ?
    `).bind(result.id).run()
    
    // 查询或创建用户
    let user = await env.DB.prepare(
      'SELECT * FROM users WHERE email = ?'
    ).bind(email).first()
    
    if (!user) {
      // 创建新用户
      const userId = Date.now().toString()
      await env.DB.prepare(`
        INSERT INTO users (id, email, created_at)
        VALUES (?, ?, ?)
      `).bind(userId, email, new Date().toISOString()).run()
      
      user = { id: userId, email }
    }
    
    // 生成 JWT Token
    const token = await new SignJWT({ userId: user.id })
      .setProtectedHeader({ alg: 'HS256' })
      .setExpirationTime('7d')
      .sign(JWT_SECRET)
    
    return Response.json({
      success: true,
      token,
      user: {
        id: user.id,
        email: user.email,
        nickname: user.nickname,
        avatar: user.avatar,
        is_vip: user.is_vip
      }
    })
  } catch (e) {
    console.error(e)
    return Response.json({ success: false, message: '登录失败' })
  }
}
```

## 八、防刷策略

### 8.1 图形验证码防刷

- 每次请求刷新验证码
- 验证码 5 分钟过期
- 使用后立即标记为已用

### 8.2 邮箱验证防刷

- 同一邮箱每日最多 3 次验证
- 验证码 10 分钟过期
- IP 限流：同一 IP 每分钟最多 5 次请求

### 8.4 IP 限流中间件

```typescript
async function rateLimit(request: Request, env: Env) {
  const ip = request.headers.get('CF-Connecting-IP') || 'unknown'
  const now = Date.now()
  const key = `rate_limit:${ip}`
  
  // 检查过去 1 分钟的请求次数
  const count = await env.KV.get(key) || 0
  
  if (count >= 5) {
    return Response.json({ 
      success: false, 
      message: '请求过于频繁，请稍后再试' 
    }, { status: 429 })
  }
  
  // 增加计数
  await env.KV.put(key, (count + 1).toString(), { expirationTtl: 60 })
}
```

## 十、优缺点分析

### ✅ 优点

- **零成本**：邮件服务免费额度足够用
- **防刷好**：图形验证码 + 邮箱双重验证
- **无门槛**：学生都有邮箱
- **可追溯**：邮箱实名，有问题可追责

### ⚠️ 缺点
