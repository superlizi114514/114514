## 一、技术栈选型

### 前端（移动端优先）

| 模块 | 技术 | 说明 |
|------|------|------|
| 框架 | **Vue 3** + Vite | 轻量、快速、易上手 |
| UI 库 | **Vant 4** | 移动端组件库（有赞出品） |
| 状态管理 | Pinia | Vue3 官方推荐 |
| 路由 | Vue Router 4 | 单页应用路由 |
| HTTP | Axios | 请求拦截、响应处理 |
| 样式 | Tailwind CSS | 原子化 CSS，快速开发 |
| 打包 | Vite | 极速构建 |

### 后端

| 模块 | 技术 | 说明 |
|------|------|------|
| 框架 | **Node.js** + Express | 轻量、高性能 |
| 语言 | TypeScript | 类型安全 |
| 数据库 | MySQL 8.0 | 主数据库 |
| 缓存 | Redis | 会话、验证码、热点数据 |
| ORM | Prisma | 类型安全的数据库操作 |
| 认证 | JWT | Token 认证 |
| 短信 | 阿里云短信 / 腾讯云短信 | 验证码发送 |

### 部署

| 模块 | 技术 | 说明 |
|------|------|------|
| 服务器 | 阿里云轻量应用服务器 | 2 核 4G 起步 |
| Web 服务器 | Nginx | 反向代理、静态资源 |
| 进程管理 | PM2 | Node.js 进程管理 |
| 存储 | 阿里云 OSS | 图片存储 |
| CDN | 阿里云 CDN | 静态资源加速 |
| HTTPS | Let's Encrypt | 免费 SSL 证书 |

## 三、API 设计

### 认证相关

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/auth/send-code` | POST | 发送短信验证码 |
| `/api/auth/login` | POST | 手机号 + 验证码登录 |
| `/api/auth/logout` | POST | 登出 |
| `/api/auth/userinfo` | GET | 获取用户信息 |

### 人员点评

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/profiles` | GET | 获取人员列表 |
| `/api/profiles` | POST | 添加人员 |
| `/api/profiles/:id` | PUT | 编辑人员 |
| `/api/profiles/:id` | DELETE | 删除人员 |
| `/api/profiles/:id/reviews` | GET | 获取点评列表 |
| `/api/profiles/:id/reviews` | POST | 添加点评 |

### 商家点评

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/merchants` | GET | 获取商家列表 |
| `/api/merchants` | POST | 添加商家 |
| `/api/merchants/:id` | GET | 获取商家详情 |
| `/api/merchants/:id/reviews` | GET | 获取点评列表 |
| `/api/merchants/:id/reviews` | POST | 添加点评 |

### 榜单

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/rankings/total` | GET | 总黑红榜（人员 + 商家） |
| `/api/rankings/profile` | GET | 个人黑红榜 |
| `/api/rankings/merchant` | GET | 商家黑红榜 |

### 搜索

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/search` | GET | 搜索人员/商家 |

### 用户

| 接口 | 方法 | 说明 |

## 六、安全方案

### 1. 短信防刷

- 同一手机号每日最多 10 次
- 60 秒内只能发送 1 次
- IP 限流：同一 IP 每分钟最多 20 次

### 2. 内容安全

- 敏感词过滤（正则 + 词库）
- 图片鉴黄（阿里云内容安全 API）
- 用户举报机制

### 3. 数据安全

- HTTPS 加密传输
- JWT Token 认证
- SQL 注入防护（Prisma 参数化查询）
- XSS 防护（输入过滤）
