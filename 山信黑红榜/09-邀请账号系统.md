# 🎫 山信黑红榜 — 邀请账号系统

---

## 一、需求概述

**场景：**
- 管理员创建邀请账号（无需邮箱验证）
- 用于定向邀请（如：学生会成员、商家合作伙伴）
- 账号密码登录，便捷高效

**流程：**
```
管理员后台：
1. 进入"邀请管理"页面
2. 点击"创建邀请账号"
3. 输入用户名、密码（可选：手机号/备注）
4. 选择账号类型（普通/商家/VIP）
5. 设置过期时间（可选）
6. 创建成功，生成邀请码/链接

用户使用：
1. 收到邀请码/链接
2. 登录页面选择"邀请账号登录"
3. 输入用户名 + 密码
4. 登录成功
```

## 二、数据库设计

### 2.1 邀请账号表

```sql
CREATE TABLE invite_accounts (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  username TEXT UNIQUE NOT NULL,
  password_hash TEXT NOT NULL,
  type TEXT DEFAULT 'normal' CHECK(type IN ('normal', 'merchant', 'vip')),
  phone TEXT,
  note TEXT,
  expires_at DATETIME,
  is_active INTEGER DEFAULT 1,
  created_by TEXT NOT NULL,  -- 创建者 ID（管理员）
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  last_login_at DATETIME
);

CREATE INDEX idx_invite_username ON invite_accounts(username);
CREATE INDEX idx_invite_active ON invite_accounts(is_active, expires_at);
```

### 2.2 邀请码表

```sql
CREATE TABLE invite_codes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  code TEXT UNIQUE NOT NULL,
  account_id INTEGER,
  type TEXT DEFAULT 'account' CHECK(type IN ('account', 'register')),
  max_uses INTEGER DEFAULT 1,
  used_count INTEGER DEFAULT 0,
  expires_at DATETIME,
  is_active INTEGER DEFAULT 1,
  created_by TEXT NOT NULL,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (account_id) REFERENCES invite_accounts(id)
);

CREATE INDEX idx_invite_code ON invite_codes(code);
```

### 2.3 用户表新增字段

```sql
ALTER TABLE users ADD COLUMN invite_code TEXT;
ALTER TABLE users ADD COLUMN account_type TEXT DEFAULT 'email';
-- account_type: 'email' | 'invite' | 'phone'
```

## 三、API 设计

### 3.1 创建邀请账号（后台接口）

```typescript
// POST /api/admin/invite/create
// 需要管理员权限

async function handleCreateInviteAccount(request: Request, env: Env) {
  const { username, password, type, phone, note, expireDays } = await request.json()
  
  // 验证必填项
  if (!username || !password) {
    return Response.json({ 
      success: false, 
      message: '用户名和密码必填' 
    })
  }
  
  // 检查用户名是否已存在
  const existing = await env.DB.prepare(
    'SELECT id FROM invite_accounts WHERE username = ?'
  ).bind(username).first()
  
  if (existing) {
    return Response.json({ 
      success: false, 
      message: '用户名已存在' 
    })
  }
  
  // 密码加密
  const passwordHash = await hashPassword(password)
  
  // 计算过期时间
  const expiresAt = expireDays 
    ? new Date(Date.now() + expireDays * 24 * 60 * 60 * 1000).toISOString()
    : null
  
  // 获取管理员 ID
  const adminId = await getAdminIdFromToken(request)
  
  // 创建账号
  const result = await env.DB.prepare(`
    INSERT INTO invite_accounts 
    (username, password_hash, type, phone, note, expires_at, created_by)
    VALUES (?, ?, ?, ?, ?, ?, ?)
  `).bind(username, passwordHash, type, phone, note, expiresAt, adminId).run()
  
  // 生成邀请码（可选）
  const inviteCode = generateInviteCode(8)
  await env.DB.prepare(`
    INSERT INTO invite_codes (code, account_id, type, created_by)
    VALUES (?, ?, 'account', ?)
  `).bind(inviteCode, result.lastRowId, adminId).run()
  
  return Response.json({
    success: true,
    data: {
      id: result.lastRowId,
      username,
      password,  // 明文密码，用于分享给用户
      inviteCode,
      type,
      expiresAt
    }
  })
}

// 生成邀请码
function generateInviteCode(length: number = 8): string {
  const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789'
  let result = ''
  for (let i = 0; i < length; i++) {
    result += chars.charAt(Math.floor(Math.random() * chars.length))
  }
  return result
}
```

### 3.2 批量创建邀请账号

```typescript
// POST /api/admin/invite/batch-create

async function handleBatchCreate(request: Request, env: Env) {
  const { accounts } = await request.json()
  // accounts: [{ username, password, type, phone, note, expireDays }]
  
  const results = []
  
  for (const acc of accounts) {
    // 复用单个创建逻辑
    const result = await createInviteAccount(acc, env, request)
    results.push(result)
  }
  
  return Response.json({
    success: true,
    created: results.filter(r => r.success).length,
    failed: results.filter(r => !r.success).length,
    details: results
  })
}
```

### 3.3 邀请账号登录

```typescript
// POST /api/invite/login

async function handleInviteLogin(request: Request, env: Env) {
  try {
    const { username, password } = await request.json()
    
    // 查询邀请账号
    const account = await env.DB.prepare(`
      SELECT * FROM invite_accounts 
      WHERE username = ? AND is_active = 1
    `).bind(username).first()
    
    if (!account) {
      return Response.json({ 
        success: false, 
        message: '账号不存在或已停用' 
      })
    }
    
    // 检查过期时间
    if (account.expires_at && new Date(account.expires_at) < new Date()) {
      return Response.json({ 
        success: false, 
        message: '账号已过期' 
      })
    }
    
    // 验证密码
    const valid = await verifyPassword(password, account.password_hash)
    if (!valid) {
      return Response.json({ 
        success: false, 
        message: '密码错误' 
      })
    }
    
    // 更新最后登录时间
    await env.DB.prepare(`
      UPDATE invite_accounts SET last_login_at = ? WHERE id = ?
    `).bind(new Date().toISOString(), account.id).run()
    
    // 查询或创建关联用户
    let user = await env.DB.prepare(
      'SELECT * FROM users WHERE invite_username = ?'
    ).bind(username).first()
    
    if (!user) {
      const userId = Date.now().toString()
      await env.DB.prepare(`
        INSERT INTO users (id, invite_username, account_type, created_at)
        VALUES (?, ?, 'invite', ?)
      `).bind(userId, username, new Date().toISOString()).run()
      
      user = { id: userId }
    }
    
    // 生成 JWT Token
    const token = await new SignJWT({ 
      userId: user.id, 
      accountType: 'invite',
      type: account.type
    })
      .setProtectedHeader({ alg: 'HS256' })
      .setExpirationTime('7d')
      .sign(JWT_SECRET)
    
    return Response.json({
      success: true,
      token,
      user: {
        id: user.id,
        username,
        type: account.type,
        is_vip: account.type === 'vip'
      }
    })
  } catch (e) {
    console.error(e)
    return Response.json({ success: false, message: '登录失败' })
  }
}
```

### 3.4 邀请码登录（可选）

```typescript
// POST /api/invite/code-login

async function handleInviteCodeLogin(request: Request, env: Env) {
  const { code } = await request.json()
  
  // 查询邀请码
  const inviteCode = await env.DB.prepare(`
    SELECT * FROM invite_codes 
    WHERE code = ? AND is_active = 1 AND used_count < max_uses
  `).bind(code).first()
  
  if (!inviteCode) {
    return Response.json({ 
      success: false, 
      message: '邀请码无效或已过期' 
    })
  }
  
  // 检查过期
  if (inviteCode.expires_at && new Date(inviteCode.expires_at) < new Date()) {
    return Response.json({ 
      success: false, 
      message: '邀请码已过期' 
    })
  }
  
  // 如果是账号类型邀请码，获取账号信息
  let account = null
  if (inviteCode.account_id) {
    account = await env.DB.prepare(
      'SELECT * FROM invite_accounts WHERE id = ?'
    ).bind(inviteCode.account_id).first()
  }
  
  // 增加使用次数
  await env.DB.prepare(`
    UPDATE invite_codes SET used_count = used_count + 1 WHERE id = ?
  `).bind(inviteCode.id).run()
  
  return Response.json({
    success: true,
    account: account ? { username: account.username, type: account.type } : null,
    inviteCode: code
  })
}
```

### 3.5 获取邀请账号列表（后台）

```typescript
// GET /api/admin/invite/list

async function handleGetInviteList(request: Request, env: Env) {
  const url = new URL(request.url)
  const page = parseInt(url.searchParams.get('page') || '1')
  const pageSize = parseInt(url.searchParams.get('pageSize') || '20')
  const type = url.searchParams.get('type')
  const status = url.searchParams.get('status')
  
  let whereClause = '1=1'
  const bindings: any[] = []
  
  if (type) {
    whereClause += ' AND type = ?'
    bindings.push(type)
  }
  
  if (status === 'active') {
    whereClause += ' AND is_active = 1 AND (expires_at IS NULL OR expires_at > ?)'
    bindings.push(new Date().toISOString())
  } else if (status === 'expired') {
    whereClause += ' AND (expires_at IS NOT NULL AND expires_at <= ?)'
    bindings.push(new Date().toISOString())
  }
  
  const offset = (page - 1) * pageSize
  
  const accounts = await env.DB.prepare(`
    SELECT * FROM invite_accounts 
    WHERE ${whereClause}
    ORDER BY created_at DESC
    LIMIT ? OFFSET ?
  `).bind(...bindings, pageSize, offset).all()
  
  const total = await env.DB.prepare(`
    SELECT COUNT(*) as count FROM invite_accounts WHERE ${whereClause}
  `).bind(...bindings).first()
  
  return Response.json({
    success: true,
    data: accounts,
    pagination: {
      page,
      pageSize,
      total: total.count
    }
  })
}
```

### 3.6 管理邀请账号（停用/删除/续期）

```typescript
// POST /api/admin/invite/update

async function handleUpdateInvite(request: Request, env: Env) {
  const { id, action, expireDays } = await request.json()
  
  if (action === 'deactivate') {
    await env.DB.prepare(`
      UPDATE invite_accounts SET is_active = 0 WHERE id = ?
    `).bind(id).run()
  } else if (action === 'activate') {
    await env.DB.prepare(`
      UPDATE invite_accounts SET is_active = 1 WHERE id = ?
    `).bind(id).run()
  } else if (action === 'extend' && expireDays) {
    const newExpires = new Date(Date.now() + expireDays * 24 * 60 * 60 * 1000).toISOString()
    await env.DB.prepare(`
      UPDATE invite_accounts SET expires_at = ? WHERE id = ?
    `).bind(newExpires, id).run()
  } else if (action === 'delete') {
    await env.DB.prepare(`
      DELETE FROM invite_accounts WHERE id = ?
    `).bind(id).run()
  }
  
  return Response.json({ success: true })
}
```

## 六、安全注意事项

### 6.1 密码安全

- 密码使用 bcrypt 加密存储
- 创建时显示明文密码（一次性）
- 建议用户首次登录后修改密码

### 6.2 权限控制

- 仅管理员可创建邀请账号
- 操作日志记录（谁创建、何时创建）
- 敏感操作二次验证

### 6.3 防滥用

// GET /api/admin/invite/export

async function handleExport(request: Request, env: Env) {
  const accounts = await env.DB.prepare(`
    SELECT username, type, phone, note, expires_at, created_at, last_login_at
    FROM invite_accounts
    ORDER BY created_at DESC
  `).all()
  
  // 生成 CSV
  const csv = [
    ['用户名', '类型', '手机号', '备注', '过期时间', '创建时间', '最后登录'],
    ...accounts.map(a => [
      a.username,
      a.type,
      a.phone || '',
      a.note || '',
      a.expires_at || '永不过期',
      a.created_at,
      a.last_login_at || '未登录'
    ])
  ].map(row => row.join(',')).join('\n')
  
  return new Response(csv, {
    headers: {
      'Content-Type': 'text/csv',
      'Content-Disposition': 'attachment; filename="invite_accounts.csv"'
    }
  })
}
