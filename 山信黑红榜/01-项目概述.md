# 🏆 山信黑红榜 — 项目概述

## 一、产品定位

校园匿名点评平台，用户可添加人员/商家进行红榜（好评）或黑榜（差评）点评。

## 二、核心特色

- ✅ **手机号登录**：短信验证码登录，安全可靠
- ✅ **移动端优先**：响应式设计，完美适配手机
- ✅ **自由添加**：用户自己添加想点评的人或商家
- ✅ **真正匿名**：付费也无法破解匿名，保护隐私
- ✅ **双轨点评**：人员点评 + 商家点评
- ✅ **每日限次**：每人每天可点评 3 次

## 三、盈利模式

| 来源 | 预估月收入 | 说明 |
|------|------------|------|
| 商家认证 | ¥1500-2000 | ¥100-200/月/家 |
| 搜索置顶 | ¥1000-2000 | ¥200-400/周/家 |
| 优惠券分成 | ¥500-1000 | 核销分成 5-10% |
| 会员订阅 | ¥400-800 | ¥5-10/月 |
| 广告位 | ¥1500-3000 | Banner/插屏 |
| **合计** | **¥5000-9000/月** | （50 家商家 + 1000 日活） |

## 四、服务器成本

| 阶段 | 配置 | 月成本 |
|------|------|--------|
| 初期 | 2 核 4G 3Mbps | ¥80-150 |
| 成长期 | 4 核 8G 5Mbps | ¥300-500 |
| 成熟期 | 8 核 16G 10Mbps | ¥800-1500 |

## 五、技术栈建议

| 模块 | 推荐方案 |
|------|----------|
| 前端 | Vue3 / React + TypeScript |
| 后端 | Node.js / Python / Go |
| 数据库 | MySQL + Redis |
| 部署 | 阿里云/腾讯云轻量服务器 |
| 存储 | OSS（图片）+ CDN |

## 六、开发周期

| 阶段 | 时间 | 内容 |
|------|------|------|
| 第一阶段 | 2-3 周 | 基础点评功能（人员 + 商家） |
| 第二阶段 | 1-2 周 | 商家认领 + 推广服务 |
| 第三阶段 | 1-2 周 | 优惠券 + 数据分析 |
