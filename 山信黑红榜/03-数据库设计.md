# 🗄️ 山信黑红榜 — 数据库设计

---

## 一、用户表 (users)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(32) | 用户 ID（User_xxx） |
| nickname | VARCHAR(50) | 昵称 |
| phone | VARCHAR(20) | 绑定手机（可选） |
| device_id | VARCHAR(64) | 设备指纹 |
| is_vip | BOOLEAN | 是否会员 |
| vip_expire | DATETIME | 会员过期时间 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

---

## 二、人员档案表 (profiles)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| user_id | VARCHAR(32) | 添加者 ID |
| name | VARCHAR(50) | 姓名 |
| campus | VARCHAR(50) | 校区 |
| class | VARCHAR(50) | 班级 |
| is_public | BOOLEAN | 是否公开 |
| created_at | DATETIME | 创建时间 |

**索引：** `(user_id, name)` 唯一索引

---

## 三、商家表 (merchants)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| name | VARCHAR(100) | 商家名称 |
| category | VARCHAR(50) | 分类 |
| address | VARCHAR(200) | 地址 |
| phone | VARCHAR(20) | 电话 |
| hours | VARCHAR(100) | 营业时间 |
| avg_price | DECIMAL(10,2) | 人均消费 |
| cover_image | VARCHAR(255) | 封面图 |
| is_certified | BOOLEAN | 是否认证 |
| certified_at | DATETIME | 认证时间 |
| created_at | DATETIME | 创建时间 |

---

## 四、人员点评表 (profile_reviews)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| reviewer_id | VARCHAR(32) | 点评者 ID |
| profile_id | INT | 被点评人 ID |
| type | ENUM('red','black') | 红榜/黑榜 |
| content | TEXT | 点评内容 |
| is_anonymous | BOOLEAN | 是否匿名 |
| like_count | INT | 点赞数 |
| report_count | INT | 举报数 |
| created_at | DATETIME | 创建时间 |

**索引：** `(reviewer_id, profile_id, DATE(created_at))` 唯一索引（每日限评）

---

## 五、商家点评表 (merchant_reviews)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| reviewer_id | VARCHAR(32) | 点评者 ID |
| merchant_id | INT | 商家 ID |
| rating | TINYINT | 评分（1-5） |
| type | ENUM('red','black') | 红榜/黑榜 |
| content | TEXT | 点评内容 |
| tags | JSON | 推荐标签 |
| images | JSON | 图片 URL 数组 |
| is_anonymous | BOOLEAN | 是否匿名 |
| like_count | INT | 点赞数 |
| report_count | INT | 举报数 |
| created_at | DATETIME | 创建时间 |

---

## 六、举报表 (reports)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| review_id | INT | 点评 ID |
| review_type | ENUM('profile','merchant') | 点评类型 |
| reporter_id | VARCHAR(32) | 举报者 ID |
| reason | TEXT | 举报理由 |
| evidence | JSON | 证据图片 |
| status | ENUM('pending','approved','rejected') | 状态 |
| handler_id | VARCHAR(32) | 处理者 ID |
| created_at | DATETIME | 创建时间 |

---

## 七、推广订单表 (promotion_orders)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| merchant_id | INT | 商家 ID |
| service_type | ENUM('certified','top_search','banner','coupon') | 服务类型 |
| price | DECIMAL(10,2) | 金额 |
| start_date | DATE | 开始日期 |
| end_date | DATE | 结束日期 |
| status | ENUM('pending','active','expired') | 状态 |
| created_at | DATETIME | 创建时间 |

---

## 八、优惠券表 (coupons)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| merchant_id | INT | 商家 ID |
| name | VARCHAR(100) | 券名称 |
| discount | DECIMAL(10,2) | 优惠金额 |
| min_spend | DECIMAL(10,2) | 最低消费 |
| total_count | INT | 发放数量 |
| used_count | INT | 已核销数量 |
| expire_date | DATE | 过期日期 |
| created_at | DATETIME | 创建时间 |

---

## 九、优惠券核销表 (coupon_redemptions)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| coupon_id | INT | 优惠券 ID |
| user_id | VARCHAR(32) | 使用者 ID |
| redeemed_at | DATETIME | 核销时间 |
| order_amount | DECIMAL(10,2) | 订单金额 |

## 十、管理员表 (admins)
