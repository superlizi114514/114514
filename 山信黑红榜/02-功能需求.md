## 一、登录系统（手机号验证码）

### 1.1 手机号登录
- [ ] 输入手机号（11 位校验）
- [ ] 获取短信验证码（60 秒倒计时）
- [ ] 输入 6 位验证码登录
- [ ] 自动注册（首次登录即注册）
- [ ] 登录状态持久化（7 天）

### 1.2 用户信息
- [ ] 登录后设置昵称（可选）
- [ ] 上传头像（可选）
- [ ] 绑定设备（多设备管理）

### 1.3 安全机制
- [ ] 短信验证码防刷（同一手机号每日最多 10 次）
- [ ] 登录失败 5 次锁定 30 分钟
- [ ] 设备指纹识别
- [ ] IP 限流（每分钟最多 10 次请求）
- [ ] 敏感操作二次验证（修改密码、解绑手机）

## 二、人员点评模块

### 2.1 添加人员
- [ ] 姓名（必填）
- [ ] 校区（可选）
- [ ] 班级（可选）
- [ ] 仅自己可见（私有列表）
- [ ] 去重校验（同一用户不能重复添加）

### 2.2 点评人员
- [ ] 从我的列表选择对象
- [ ] 红榜/黑榜二选一
- [ ] 点评理由（10-200 字，必填）
- [ ] 匿名/实名发布
- [ ] 每日限次：3 次/天
- [ ] 同一人每天只能被同一用户点评 1 次

### 2.3 查看榜单
- [ ] 搜索姓名查看红黑记录
- [ ] 按校区/班级筛选
- [ ] 红榜/黑榜分开显示
- [ ] 点评详情：时间、点评者（匿名/实名）、理由
