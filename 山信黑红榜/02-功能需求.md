## 一、登录系统（匿名化）

### 1.1 匿名登录
- [ ] 生成随机用户 ID（格式：User_xxx）
- [ ] 可自定义昵称（可选）
- [ ] 登录状态本地存储（LocalStorage）
- [ ] 设备 ID 绑定（防刷）

### 1.2 可选绑定
- [ ] 绑定手机号 → 数据云端同步
- [ ] 多设备登录支持

### 1.3 安全机制
- [ ] 登录失败 5 次锁定 30 分钟
- [ ] 设备指纹识别
- [ ] IP 限流（每分钟最多 10 次请求）

## 二、人员点评模块

### 2.1 添加人员
- [ ] 姓名（必填）
- [ ] 校区（可选）
- [ ] 班级（可选）
- [ ] 仅自己可见（私有列表）
- [ ] 去重校验（同一用户不能重复添加）

### 2.2 点评人员
- [ ] 从我的列表选择对象
- [ ] 红榜/黑榜二选一
- [ ] 点评理由（10-200 字，必填）
- [ ] 匿名/实名发布
- [ ] 每日限次：3 次/天
- [ ] 同一人每天只能被同一用户点评 1 次

### 2.3 查看榜单
- [ ] 搜索姓名查看红黑记录
- [ ] 按校区/班级筛选
- [ ] 红榜/黑榜分开显示
- [ ] 点评详情：时间、点评者（匿名/实名）、理由
